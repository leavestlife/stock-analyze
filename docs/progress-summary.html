Project Summary · 2026-05-22

CAN SLIM 종목 분석기 진행 요약

레퍼런스 사이트를 기반으로 실데이터 스캐너, 상세 팝업, 엔트리 점수, 포트폴리오 리스크 관리까지 확장한 현재 상태를 정리했습니다.

현재 상태: 실행 가능
데이터 연결 FMP · Alpha · DART

무료 API와 캐시 기반으로 가격, 재무, 뉴스, 공시를 보강합니다.

점수 체계 Total + Entry

좋은 종목과 좋은 진입 타이밍을 분리해서 봅니다.

리스크 엔진 ATR · R:R · Heat

손절 기준 오픈 리스크와 포트폴리오 과열도를 계산합니다.

검증 상태 11/11 통과

서버 계약 테스트와 브라우저 스모크 검증을 통과했습니다.

완료된 주요 작업

STEP 1
레퍼런스 UI 구현

섹터 사이드바, 종목 요약 테이블, 상세 팝업, CAN SLIM/기술/재무/인사이트 탭을 구성했습니다.

STEP 2
실데이터 연결

FMP, Alpha Vantage, Yahoo/Naver 가격 캐시, DART/SEC 링크를 연결하고 데이터 신뢰도 상태를 표시했습니다.

STEP 3
점수 계산식 정렬

캡처된 CAN SLIM, 퀀트, 수학 지표의 raw score, normalized score, contribution 구조를 맞췄습니다.

STEP 4
엔트리 스코어 고도화

단순 점수 대신 상승추세, RSI 눌림, VWAP, 돌파 거래량, 변동성, 추격 위험을 함께 보는 EntryScore v2를 적용했습니다.

STEP 5
트레이딩 스킬 반영

position sizing, breakout planning, VCP gate 개념을 적용해 권장 수량, 손절폭, R:R, 피벗 상태를 표시했습니다.

STEP 6
포트폴리오 관리 기능

보유 종목 추가/수정/삭제, 현재가 평가, 손익, 손절 기준 오픈 리스크, Heat, 섹터 집중도, CSV 내보내기/불러오기를 추가했습니다.

현재 포트폴리오 기능

기능 상태 설명
보유 종목 추가/수정/삭제 완료 티커, 수량, 평균단가를 입력하고 브라우저 로컬 저장소에 저장합니다.
현재가 평가 완료 기존 /api/stocks/:ticker 상세 데이터를 재사용합니다.
오픈 리스크 완료 현재가와 서버 계산 손절가 차이를 기준으로 예상 손실을 계산합니다.
포트폴리오 Heat 완료 통화별 오픈 리스크를 계좌 규모 대비 비율로 표시합니다.
섹터 집중도 완료 평가금액 기준 가장 큰 섹터 비중을 보여줍니다.
CSV 내보내기/불러오기 완료 accountSize, ticker, shares, avgCost 형식입니다.
