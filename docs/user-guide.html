종목 분석기 사용 가이드

이 앱은 관심 종목을 먼저 관리하고, 각 종목의 점수와 진입 타점, 데이터 신뢰도를 함께 확인하는 베타 분석 도구입니다.

1. 첫 화면: 관심 종목

왼쪽 최상단의 관심 종목 영역이 기본 시작 화면입니다. 입력창에 AAPL, NVDA, 005930처럼 티커를 넣고 추가하면 목록에 들어갑니다.

AAPLNVDAGOOGLQQQM 같은 빠른 추가 버튼도 사용할 수 있습니다.

2. 데이터 신뢰도 읽는 법

  • 높음: 가격과 핵심 재무/공시 출처가 대부분 연결된 상태입니다.
  • 보통: 일부 데이터가 부분 연결이거나 대체 계산이 섞인 상태입니다.
  • 낮음: 가격 외 핵심 출처가 부족하거나 교차검증 경고가 있는 상태입니다.

관심 종목 카드 아래에는 종목별로 약한 출처가 표시됩니다. 예: FMP, Alpha, SEC.

3. 데이터 보강 상태

상단의 자동 보강 영역은 실제 서버 보강 큐 상태를 보여줍니다.

  • 우선 대기: 관심 종목이라 다른 종목보다 먼저 처리됩니다.
  • 실제 보강 중: 서버가 가격/재무/공시 데이터를 가져와 계산 중입니다.
  • 보강 완료: 보강 결과가 캐시에 반영되었습니다.

4. 상세 팝업

종목을 클릭하면 상세 팝업이 열립니다. 상단의 데이터 신뢰도 배너에서 이 종목 점수가 어느 정도 믿을 만한지 먼저 확인하세요.

신뢰도 낮음이면 점수 자체보다 약한 출처와 보강 필요 항목을 먼저 보는 것이 좋습니다.

5. 점수 사용 원칙

  1. TotalScore는 종목 자체의 품질과 시장 내 강도를 봅니다.
  2. EntryScore는 지금 들어갈 만한 자리인지 봅니다.
  3. 점수가 높아도 신뢰도가 낮으면 매수 판단보다 데이터 보강을 먼저 확인하세요.
  4. 이 앱은 매수 버튼이 아니라 후보군 압축기입니다.

6. 추천 워크플로우

  1. 관심 종목에 실제로 보는 티커를 추가합니다.
  2. 왼쪽 카드에서 신뢰도와 보강필요 수를 확인합니다.
  3. 자동 보강이 끝난 뒤 목록 점수와 시그널을 봅니다.
  4. 종목을 클릭해 상세 팝업에서 데이터 출처, 차트, CAN SLIM, 기술/재무 지표를 확인합니다.
  5. 포트폴리오에 넣었다면 손절가, Heat, 오픈 리스크를 함께 봅니다.
